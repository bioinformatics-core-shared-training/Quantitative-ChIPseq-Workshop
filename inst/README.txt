If you are not running a docker image with the data installed into inst/extdata:

-Download the data from https://content.cruk.cam.ac.uk/bioinformatics/software/DiffBind/DiffBind_vignette_data.tar.gz 

-unzip and untar the data:      tar xzf DiffBind_vignette_data.tar.gz
-Move the data to inst/extdata: mv DiffBind_Vignette/ inst/extdata/ 
-Delete the downloaded archive: rm DiffBind_vignette_data.tar.gz
